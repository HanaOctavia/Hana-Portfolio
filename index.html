Mekaar Prb-C DSS

Problems that often arise in Mekaar Prb-C are related to making decisions on loan amounts and selecting prospective borrowers. This happens because there are no clear and structured parameters to determine the decision. Based on these problems, we created a web-based platform called Mekaar Prb-C which is useful as a Decision Support System for making decisions about who is eligible to receive a loan. Mekarr Prb-C contains some information such as an introduction to DSS, an introduction to Mekaar Prb-C and a DSS calculation model using the AHP (Analytical Hierarchy Process) algorithm.

Task :

1. Perform AHP calculations using spreadsheets
2. Built an about page and an explanation page for AHP calculation steps

Tomato Disease Prediction App

Tomato is a horticultural plant that is very easy to find in Indonesia. Diseases in tomato plants are usually caused by fungi, bacteria and viruses. The symptoms can be seen from changes in the shape and color of the leaves. The development of this web app was carried out to assist farmers in identifying the types of diseases on tomato leaves.

Task :

1. Built a front-end website using ReactJS
2. Built Machine Learning Model with Convolutional Neural Network(CNN)
3. Built API with python for deploying the Machine Learning Model
